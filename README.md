## üöÄ Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## üìù Customization

### Update Your Information

1. **Hero Section** (`components/Hero.tsx`):
   - Change "Your Name" to your actual name
   - Update the description and tagline
   - Add your social media links

2. **About Section** (`components/About.tsx`):
   - Update the bio text
   - Modify the statistics

3. **Projects Section** (`components/Projects.tsx`):
   - Replace with your actual projects
   - Add real GitHub and live demo links
   - Update project descriptions and tech stacks

4. **Skills Section** (`components/Skills.tsx`):
   - Update skill categories and proficiency levels
   - Add or remove skills as needed

5. **Contact Section** (`components/Contact.tsx`):
   - Update email, phone, and location
   - Connect the form to a backend service (e.g., Formspree, EmailJS)

6. **Metadata** (`app/layout.tsx`):
   - Update the title and description

### Styling

Colors can be customized in `app/globals.css`:
- `--primary`: Main brand color
- `--secondary`: Secondary color
- `--accent`: Accent color
- `--dark`: Background color
